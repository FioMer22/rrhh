# PWA empaqueta como apk - Migración progresiva a nativo
- PHP PWA webview parte - back y front
- Android studio (kotlin) Android 8.0 Oreo - funciones nativas especificas requeridas (notificaciones nativas, acceso GPS y widget para marcación[entrada/salida])
- Mysql

# Uso
*App para RRHH*
- Permite tener usuarios, marcar hora de entrada, salida almuerzo, crear activdades registrar inicio y fin de actividades, y crear notificacione para empleados.
